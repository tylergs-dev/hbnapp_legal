# Hushabye Legal Documents

This repository hosts the public legal documents for the Hushabye mobile application.

## Documents

* **Privacy Policy**
  Describes how the app handles user data.
  → `/privacy-policy.html`

* **Terms of Service**
  Defines the terms and conditions for using the app.
  → `/terms.html`

## Purpose

These documents are published via GitHub Pages to provide publicly accessible URLs required for:

* Apple App Store submission
* Google Play Store submission
* User transparency and compliance

## Data Collection Summary

The Hushabye app does **not collect, store, or share any personal data**.
All functionality operates without user accounts, analytics, or tracking.

For full details, see the Privacy Policy.

## Updates

These documents may be updated periodically to reflect changes in the app or legal requirements.
The "Effective Date" within each document will indicate the latest revision.
